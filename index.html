About Us

CookEasy is a mobile app designed to support non-verbal learners in building basic cooking routines—starting with simple meals like breakfast. It uses PECS-style picture cards and a step-by-step task board to make each step clear and repeatable.

  • PECS-based visual task cards for guided meal prep
  • Built-in caregiver lesson plans and visual checklists
  • Distraction-free UI, designed for cognitive accessibility

CookEasy isn’t just about cooking—it’s about encouraging independence, supporting home routines, and making learning accessible for all.

Empowering Independence with CookEasy

CookEasy supports both learners and caregivers by offering visual instruction, flexible routines, and easy navigation—all grounded in assistive learning strategie

01 What is CookEasy and how does it help?

CookEasy is a mobile app designed to guide non-verbal learners through simple cooking routines using PECS-style visual cards and a step-by-step task board. It supports independent skill-building and provides structured guidance for caregivers.

02 How does CookEasy improve the cooking experience?

It breaks down routines like preparing eggs or toast into small, visual steps learners can follow independently. Caregivers get matching lesson guides and visual cards to ensure routines are consistent, repeatable, and accessible at home.

03 How does CookEasy enhance cooking independence for non-verbal individuals?

By using tap-to-move visual task boards (To Do → Doing → Done), CookEasy helps users understand progress without needing verbal instructions. The built-in caregiver module reinforces support while gradually reducing dependency.

CookEasy Features

Making meal preparation simple, accessible, and fun for non-verbal individuals with autism.

Visual Cooking Tracker

Step-by-step routine board (To Do → Doing → Done) guides users through simple meals like scrambled or hard-boiled eggs.

Caregiver Mode

Includes printable lesson plans and visual guides for structured home teaching, adapted from PDK-style methods.

PECS Visual Cards

Custom card sets for tools, actions, food, safety, emotions, and routines—designed for tap-to-track interaction and matching visuals.

Offline & Profile Access

Supports offline tracking with local data caching and personalized user login via Firebase.
